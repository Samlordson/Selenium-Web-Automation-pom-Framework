# Appium Mobile Automation Framework

This project is an Android mobile automation framework developed using Appium and Java. It follows the Page Object Model (POM) design pattern and uses TestNG for test execution with Extent Reports for reporting.

## Tech Stack

- Java
- Appium
- TestNG
- Maven
- Selenium WebDriver
- Page Object Model (POM)
- Extent Reports

## Features

- Android Mobile Automation
- Page Object Model (POM)
- Data-Driven Testing
- XML Suite Execution
- Extent Reports
- Reusable Utility Classes

## Project Structure

```
src
├── base
├── pages
├── testcases_buyer
├── testcases_seller
├── utils
└── resources
```

## Execute Tests

```bash
mvn clean test
```

## Report

After execution, the Extent Report is generated in the reports directory.
